## 快速对照表

| 场景 | 操作 | AI 自动做什么 |
|------|------|--------------|
| 想开发新功能 | 描述需求 | brainstorming → writing-plans |
| 遇到 Bug | 描述问题 | systematic-debugging |
| 多个任务并行 | 让 AI 并行处理 | dispatching-parallel-agents |
| 想验证完成 | 说"可以合并了" | verification-before-completion |

## 常用命令速查

```
# 安装
/plugin marketplace add obra/superpowers-marketplace
/plugin install superpowers@superpowers-marketplace

# 验证安装
/plugin list

# 常用对话
帮我规划这个功能：xxx
帮我修这个 Bug：xxx
这个代码审查一下
可以合并了
```

## 技能触发关键词

| 技能 | 触发方式 |
|------|----------|
| brainstorming | "帮我规划..."、"我想加..."、"设计..." |
| writing-plans | 设计确认后自动触发 |
| TDD | 任务执行时自动触发 |
| requesting-code-review | 任务完成后自动触发 |
| systematic-debugging | "修 Bug"、"出问题了"、"报错" |
| verification-before-completion | "完成"、"合并"、"结束了" |
