# Superpowers 系列教程设计文档

## 概述

将现有的 Superpowers 单篇快速入门教程，扩展为 4 篇系列教程，用于 SEO 引流。系列完全免费，付费服务定位为 1 对 1 辅导和定制化培训。

---

## 背景

现有 `docs/superpowers-quickstart.md` 是一篇 3000+ 字的中文入门教程，涵盖：
- 安装指南
- 7 步核心工作流
- 实战示例
- 付费课程引流

需求：将此单篇扩展为完整系列，提升 SEO 覆盖面和引流转化。

---

## 系列结构

### 整体架构

```
第1篇（免费） → 第2篇（免费） → 第3篇（免费） → 第4篇（免费引流）
   安装          核心原理        实战案例        进阶+付费钩子
```

### 引流逻辑

1. **前 3 篇**：给足基础价值，建立读者信任和认知
2. **第 4 篇**：展示深度和进阶能力，自然引出付费服务
3. **付费服务**：1 对 1 辅导、团队培训、技能定制

## 各篇详细设计

### 第 1 篇：安装与验证

**文件**：`docs/superpowers/01-installation.md`

**目标读者**：从未接触过 Superpowers 的新用户

**篇幅**：约 1500 字

**内容结构**：
1. 开篇痛点共鸣（3-4 个 AI 编程常见困境）
2. Superpowers 是什么（30 秒电梯演讲 + 核心理念表格）
3. 支持平台一览（表格：Claude Code / Cursor / Codex / Gemini CLI）
4. 安装步骤（Claude Code 详细，其他平台简要）
5. 验证安装成功
6. 快速体验：第一个对话演示

**SEO 关键词**：`Superpowers 安装`、`Claude Code 插件安装`、`AI 编程框架`

---

### 第 2 篇：核心工作流详解

**文件**：`docs/superpowers/02-core-workflow.md`

**目标读者**：已安装、需要深入理解原理的用户

**篇幅**：约 3000 字（含代码示例）

**内容结构**：
1. 7 步工作流全景图（Mermaid 流程图）
2. Step 1-7 各用 200-300 字详解
   - Step 1：头脑风暴（苏格拉底式提问示例）
   - Step 2：Git Worktree 隔离工作区
   - Step 3：任务规划（2-5 分钟原则）
   - Step 4：Subagent 驱动开发
   - Step 5：TDD 红绿重构循环（含代码示例）
   - Step 6：代码审查（严重程度分级报告）
   - Step 7：完成分支（合并选项）
3. TDD 铁律详解："没有先失败的测试，就不能写生产代码"
4. 为什么流程比 AI 聪明更重要

**SEO 关键词**：`AI 编程工作流`、`TDD 教程`、`AI 代码审查`

---

### 第 3 篇：实战案例——评论系统

**文件**：`docs/superpowers/03-real-world-example.md`

**目标读者**：想看真实效果、动手实践的用户

**篇幅**：约 3500 字（含完整代码）

**内容结构**：
1. 案例背景：博客评论系统需求描述
2. 完整流程演示
   - 头脑风暴阶段的提问与回答（对话节选）
   - 设计文档摘要
   - 任务规划输出示例
   - Subagent 执行片段
   - TDD 代码演示
   - 代码审查报告
3. 常见陷阱与避坑指南（3-4 条）
4. 实战效果总结（效率提升、代码质量对比）

**SEO 关键词**：`AI 开发实战`、`Superpowers 教程`、`AI 编程案例`

---

### 第 4 篇：进阶技能与团队协作（引流章节）

**文件**：`docs/superpowers/04-advanced-teams.md`

5. 下一步：付费服务（1 对 1 辅导、团队培训、技能定制）

**SEO 关键词**：`AI 团队协作`、`Superpowers 进阶`、`AI 开发培训`

## 质量标准

### 内容质量
- 每篇独立可读，无需其他篇作为前提
- 代码示例真实可运行（使用 Python 或 JavaScript/TypeScript）
- 对话示例自然流畅，符合 Superpowers 技能规范
- 中文表达清晰简洁，符合 CLAUDE.md 规范

### SEO 标准
- 每篇包含 3-5 个目标关键词
- 标题使用 H1，每篇开头包含关键词
- 结构化数据：使用表格、列表、代码块
- 内部链接：系列各篇相互引用

### 引流设计
- 前 3 篇给足价值，不强行推销
- 第 4 篇自然过渡到付费服务
- 付费服务描述具体（1 对 1 辅导 / 团队培训 / 技能定制）
- 提供明确联系方式或 CTA
