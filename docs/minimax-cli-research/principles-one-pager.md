输出流设计（专为 Agent 优化）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  传统 CLI 输出：
  ┌────────────────────────────────────────────────────────┐
  │ [████████████░░░░] 67% | Generating... | ETA: 12s     │  ← 人类友好
  │ > /tmp/output_20260409.jpg                             │  ← 数据被淹没
  └────────────────────────────────────────────────────────┘
                        ↑
                   stdout 混杂

  MMX-CLI 输出：
  ┌────────────────────────────────────────────────────────┐
  │ stderr: [████████████░░░░] 67% | Generating... | ETA  │  ← 人类可见
  │ stdout: /tmp/output_20260409.jpg                       │  ← Agent 纯净
  │         或: {"file": "/tmp/...", "id": "xxx"}           │  ← JSON 结构
  └────────────────────────────────────────────────────────┘

  Agent 模式（--quiet --output json）：
  ┌────────────────────────────────────────────────────────┐
  │ stdout: {"status":"success","file_id":"xxx","url":".."} │  ← 纯数据
  │ exit code: 0                                           │  ← 语义化
  └────────────────────────────────────────────────────────┘


语义退出码体系
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  exit 0   ████████████████████  成功

  exit 1   ██  通用错误

  exit 10  ██  鉴权失败（Token 无效/过期）
            → Agent 执行: mmx auth refresh / 更新 key

  exit 11  ██  参数错误（缺少必需参数/类型不匹配）
            → Agent 执行: 检查参数列表，补充后重试

  exit 12  ██  异步任务超时
            → Agent 执行: 使用 --async 重新提交，查 task status

  exit 13  ██  网络异常
            → Agent 执行: 指数退避重试

  exit 14  ██  配额不足（Token 耗尽）
            → Agent 执行: 暂停，提示用户充值


异步任务流程（--async）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  同步模式（默认）:
  ┌─────────────────────────────────────────────────────┐
  │ mmx video generate --prompt "..."                   │
  │              ↓                                      │
  │         等待完成（可能数分钟）                        │
  │              ↓                                      │
  │         返回结果                                    │
  └─────────────────────────────────────────────────────┘

  异步模式（--async）:
  ┌─────────────────────────────────────────────────────┐
  │ mmx video generate --prompt "..." --async          │
  │              ↓                                      │
  │      立即返回 task_id（不阻塞）                       │
  │              ↓                                      │
  │   Agent 可并行执行其他任务                            │
  │              ↓                                      │
  │   mmx video task get --task-id xxx   ← 查询进度      │
  │              ↓                                      │
  │   mmx video download --file-id yyy --out v.mp4      │
  └─────────────────────────────────────────────────────┘


双区域路由
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  mmx config set --key region --value cn
                      │
                      ▼
             ┌─────────────────┐
             │  region = "cn"  │    → api.minimaxi.com
             │  region = "global"│ → api.minimax.io
             └─────────────────┘

  API 调用时自动路由，Agent 无需感知区域差异。
