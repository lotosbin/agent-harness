# Droid 入门指导

> 快速上手 Factory AI Droid，从安装到完成第一个任务。

---

## 环境要求

- **CLI**: macOS / Linux / Windows
- **Python SDK**: Python 3.10+
- **TypeScript SDK**: Node.js 18+
- **Droid CLI**: 已安装并加入 PATH

---

## 第一步：安装 Droid CLI

```bash
# macOS / Linux
curl -fsSL https://app.factory.ai/cli | sh

# Windows
irm https://app.factory.ai/cli/windows | iex

# npm（备用）
npm install -g droid
```

安装完成后，运行 `droid --version` 确认。

---

## 第二步：启动交互式会话

```bash
# 进入项目目录
cd ~/my-project

# 启动 Droid 交互会话
droid
```

**快捷键**:
- `!` — 进入 Bash 直接模式
- `Esc` — 退出 Bash 模式，返回 Droid

---

## 第三步：Python SDK 快速开始

### 安装

```bash
pip install droid-sdk
# 或
uv add droid-sdk
```

### 简单查询（query 模式）

```python
import asyncio
from droid_sdk import query

async def main():
    async for msg in query("用 Python 实现一个快速排序"):
        print(msg)

asyncio.run(main())
```

### 进阶：多轮会话

```python
import asyncio
from droid_sdk import DroidClient, ProcessTransport

async def main():
    transport = ProcessTransport()
    client = DroidClient(transport)

    # 创建会话
    session_id = await client.initialize_session(cwd="/path/to/project")

    # 多轮对话
    await client.add_user_message("创建一个 REST API")
    async for msg in client.receive_response():
        print(msg)

    await client.close()

asyncio.run(main())
```

### 权限处理

```python
async def main():
    client = DroidClient(ProcessTransport())

    def my_permission_handler(tool_name: str, args: dict) -> bool:
        if tool_name == "Bash" and "rm -rf" in args.get("command", ""):
            return False  # 拒绝危险命令
        return True

    client.set_permission_handler(my_permission_handler)
    # ...
```

## 第四步：TypeScript SDK 快速开始

### 安装

```bash
npm install @factory/droid-sdk
```

### 简单查询

```typescript
import { query } from "@factory/droid-sdk";

async function main() {
  for await (const msg of query({ prompt: "用 TypeScript 实现一个堆排序" })) {
    console.log(msg);
  }
}

main();
```

### 多轮会话

```typescript
import { createSession } from "@factory/droid-sdk";

async function main() {
  const session = await createSession({ cwd: "/path/to/project" });

  await session.send("帮我创建一个 Express.js REST API");

  for await (const msg of session.stream()) {
    console.log(msg);
  }

  // 添加 MCP Server
  await session.addMcpServer({ command: "npx", args: ["-y", "some-mcp-server"] });

  await session.close();
}
```

### 配置选项

```typescript
const session = await createSession({
  cwd: "/path/to/project",
  modelId: "opus-4-5",           // 指定模型
  autonomyLevel: "medium",       // 自主级别
  reasoningEffort: "high",        // 推理投入
  execPath: "/usr/local/bin/droid",  // 自定义 droid 路径
});
```

## 第五步：GitHub Action 集成

在 `.github/workflows/droid-review.yml` 中：

```yaml
name: Droid Code Review
on: [pull_request]

jobs:
  review:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4

      - name: Run Droid Review
        uses: Factory-AI/droid-action@v1
        with:
          factory_api_key: ${{ secrets.FACTORY_API_KEY }}
          automatic_review: true
          review_depth: deep        # deep 或 shallow
```

### 安全扫描

```yaml
- name: Security Scan
  uses: Factory-AI/droid-action@v1
  with:
    factory_api_key: ${{ secrets.FACTORY_API_KEY }}
    automatic_security_review: true
    security_severity_threshold: high
    security_block_on_critical: true
```

## 常见问题

### Q: droid 命令找不到？
确保安装后 PATH 已更新，或使用绝对路径运行：
```bash
export PATH="$PATH:$(npm global prefix)/bin"
```

### Q: Python SDK 连接错误？
确认 `droid` CLI 已安装并在 PATH 中：
```bash
droid --version
```

### Q: 如何切换模型？
```bash
# CLI 交互模式中
/model opus-4.1
# 或
/model sonnet-4
```

### Q: 支持 MCP 服务器吗？
是的，通过 SDK：
```python
# Python
await session.add_mcp_server("path/to/mcp-server")

# TypeScript
await session.addMcpServer({ command: "npx", args: ["-y", "server"] })
```
