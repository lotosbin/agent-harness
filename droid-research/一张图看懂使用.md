## 命令对照速查

| 场景 | Python SDK | TypeScript SDK | CLI |
|------|-----------|----------------|-----|
| 初始化 | `DroidClient(ProcessTransport())` | `createSession()` | `droid` |
| 发送消息 | `add_user_message()` | `session.send()` | 直接输入 |
| 接收流 | `async for msg in receive_response()` | `for await of stream()` | 自动流式 |
| 中断 | `client.interrupt()` | `session.interrupt()` | `Ctrl+C` |
| 关闭 | `client.close()` | `session.close()` | `exit` |
| 添加 MCP | `session.add_mcp_server()` | `session.addMcpServer()` | — |
| 权限处理 | `set_permission_handler()` | `permissionHandler` | — |

## 配置速查

```python
# Python
client = DroidClient(transport)
client.set_permission_handler(my_handler)
session_id = await client.initialize_session(cwd="/project")
```

```typescript
// TypeScript
const session = await createSession({
  cwd: "/project",
  modelId: "opus-4.1",
  autonomyLevel: "medium",
  reasoningEffort: "high",
});
```

## GitHub Action 配置

```yaml
factory_api_key: ${{ secrets.FACTORY_API_KEY }}   # 必填
github_token: ${{ secrets.GITHUB_TOKEN }}          # 可选
automatic_review: true                              # 自动审查
review_depth: deep                                  # deep | shallow
review_model: gpt-5.2                               # 覆盖模型
automatic_security_review: true
security_severity_threshold: medium
security_block_on_critical: true
```
