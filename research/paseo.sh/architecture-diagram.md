## 数据流：一次完整的远程 agent 交互

```
  开发者 (手机)                  Relay                    本地机器
       │                          │                          │
       │  1. 打开 Paseo App       │                          │
       │─────────────────────────▶│                          │
       │                          │                          │
       │  2. 扫描 QR / 登录        │                          │
       │─────────────────────────▶│                          │
       │                          │                          │
       │  3. 建立 WebSocket        │                          │
       │◀─────────────────────────│                          │
       │                          │  4. 启动 WebSocket       │
       │                          │◀─────────────────────────│
       │                          │                          │
       │  5. 发送任务              │                          │
       │─────────────────────────▶│  6. 转发任务              │
       │                          │─────────────────────────▶│
       │                          │                          │
       │                          │     7. 启动 Claude Code   │
       │                          │◀─────────────────────────│
       │                          │                          │
       │  8. 实时输出流            │                          │
       │◀─────────────────────────│  9. 转发输出              │
       │                          │◀─────────────────────────│
       │                          │                          │
       │  ✅ 完成                  │                          │
       │                          │                          │
```

## 隐私安全模型

```
┌─────────────────────────────────────────────────────────────────────┐
│                        隐私边界                                      │
│                                                                      │
│   ┌─────────────────────────────────────────────────────────────┐   │
│   │  🔒 留在本地                                                  │   │
│   │                                                              │   │
│   │  ✅ 代码内容                                                  │   │
│   │  ✅ API Keys / Secrets                                       │   │
│   │  ✅ Skills 配置                                              │   │
│   │  ✅ MCP Servers                                              │   │
│   │  ✅ 文件系统                                                  │   │
│   │  ✅ Git 历史                                                 │   │
│   └─────────────────────────────────────────────────────────────┘   │
│                                  │                                   │
│                                  │ 只传输                            │
│                                  ▼                                   │
│   ┌─────────────────────────────────────────────────────────────┐   │
│   │  📤 通过 Relay 传输                                           │   │
│   │                                                              │   │
│   │  ⚠️ Agent 文字输出（用户可见）                                │   │
│   │  ⚠️ 指令文本（用户发送的命令）                                │   │
│   │                                                              │   │
│   │  🔒 Relay 无法解密（E2E 加密）                               │   │
│   └─────────────────────────────────────────────────────────────┘   │
│                                                                      │
└──────────────────────────────────────────────────────────────────────┘
```
