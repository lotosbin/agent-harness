## ✨ 核心功能

### 1. 全平台覆盖 🖥️📱💻

- macOS 桌面 App
- iOS / Android App
- Web 版本（浏览器直接用）
- CLI 终端

一个界面，管理所有 coding agent！

### 2. 远程控制 + E2E 加密 🔒

代码永远不会离开你的机器 💪

远程连接走的是端到端加密隧道，Relay 服务器根本看不到你的代码！

### 3. Git Worktree 隔离 🌳

同时跑多个 agent？
每个 agent 在独立 Git 分支上运行，互不干扰！

### 4. 本地语音控制 🎤

完全本地化的语音输入，不用联网！
对着手机说"帮我写个登录功能"，agent 就开始干活了~

---

## 🚀 快速上手

```bash
# 1. 安装 CLI
npm install -g @getpaseo/cli

# 2. 启动（显示 QR 码）
paseo

# 3. 手机扫码连接
```

然后就可以用手机控制 agent 了！
