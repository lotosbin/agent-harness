## 使用流程速查

```
第1步: 安装
        │
        ▼
第2步: 启动（选择客户端）
        │
        ├──▶ 终端 ──▶ paseo
        ├──▶ 桌面 ──▶ Electron App
        ├──▶ 手机 ──▶ 扫描 QR 码
        └──▶ Web ──▶ app.paseo.sh

第3步: 选择 Agent Provider
        │
        ├──▶ Claude Code ──▶ --provider claude
        ├──▶ Codex ──▶ --provider codex
        └──▶ 其他 ──▶ --provider opencode/copilot/pi

第4步: 运行任务
        │
        ├──▶ 直接运行 ──▶ paseo run "task"
        ├──▶ 隔离分支 ──▶ --worktree feature-x
        └──▶ 附加查看 ──▶ attach <id>

第5步: 监控与交互
        │
        ├──▶ 实时输出 ──▶ attach <id>
        ├──▶ 发送消息 ──▶ send <id> "message"
        └──▶ 列出所有 ──▶ ls
```
